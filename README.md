### 로그인 기능 구현

### 1. 이메일을 통한 회원가입 기능 구현하기

회원가입 폼에서 아이디, 패스워드, 이메일 정보를 입력 받는다

입력받은 이메일 주소로 확인 메일에 토큰을 담아 전송한다

회원이 수신한 메일을 통해 토큰과 함께 요청을 전송하고, 이를 validate한다
